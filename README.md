# Vefforritun 2 — Hópverkefni 2

## Hópmeðlimir

- Viktor Blær

## Uppsetning

### Kröfur

- Node.js 18+
- npm

### Keyrsla
```bash
npm install
cp .env.local.example .env.local
npm run dev
```

Opnaðu [http://localhost:3001](http://localhost:3001) í vafra.

## Umhverfisbreytur

| Breyta | Lýsing | Dæmi |
|--------|--------|------|
| `NEXT_PUBLIC_API_URL` | Slóð á bakenda API | `http://localhost:3000` |

## Admin innskráning

- Lykilorð: `admin123`
